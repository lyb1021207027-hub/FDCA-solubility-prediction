# Data Dictionary

This project uses nine molecular or simulation-derived descriptors as input variables.

| Input column in Excel | Internal model feature |
|---|---|
| Diffusion coefficient | f0 |
| CN(FDCA–H) | f1 |
| CN(FDCA–O) | f2 |
| MEPS minimal | f3 |
| MEPS maximal | f4 |
| Polarity difference | f5 |
| Solvation energy | f6 |
| δH | f7 |
| δT | f8 |

The validation file also contains:

| Output column in Excel | Meaning |
|---|---|
| Y | True target value |

Generated by the script:

| Output column | Meaning |
|---|---|
| Y_pred | Model-predicted target value |
| Residual | Y - Y_pred |
